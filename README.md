# SupervisorMatch

Simple launch guide for frontend and backend.

## 1) Run Frontend

From project root:

```bash
npm install
npm run dev
```

Frontend URL:
- http://localhost:5173

## 2) Run Backend

From project root:

```bash
cd backend
npm install
npm run start:dev
```

Backend URL:
- http://localhost:4000

## 3) Demo Accounts

- Student: `student@test.com` / `demo123`
- Supervisor: `johnson@university.edu` / `demo123`

## 4) Notes

- Frontend currently works in local prototype mode (localStorage context).
- Backend is implemented and can be tested independently.
- Full backend explanation is in `docs/BACKEND_DEFENCE_GUIDE.md`.
